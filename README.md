## 🧠 Diabetes Prediction using Multi-Algorithm Kernel Benchmarking

This project presents a systematic comparative study of **15 machine learning algorithms**, evaluated using **5 different kernel functions** and **3 data splitting strategies** for the task of diabetes prediction.

The goal of this research is to analyze how different combinations of algorithms, kernels, and data partitioning techniques influence model performance in a healthcare prediction setting.

### 🔍 Key Highlights

* Evaluation of 15 diverse ML algorithms
* Comparison across 5 kernel functions
* Analysis using 3 data split strategies (e.g., train-test, cross-validation)
* Performance metrics comparison (accuracy, precision, recall, F1-score, etc.)
* Focused application on diabetes prediction datasets

### 🎯 Objective

To identify the most effective algorithm–kernel–data split combination for accurate and reliable diabetes prediction.

### 📊 Outcome

This study provides insights into model selection and optimization strategies for medical prediction tasks, particularly in diabetes diagnosis.
